## 🌟 About

PathFinder AI is an intelligent AI-powered career platform designed to help students, developers, and professionals:

- Build ATS-friendly resumes  
- Generate personalized cover letters  
- Prepare for interviews using AI  
- Get career insights & skill trends  

---

## ✨ Features

- 📄 AI Resume Builder (ATS Optimized)  
- ✉️ Cover Letter Generator  
- 🎯 Interview Preparation System  
- 📊 Industry Insights & Trends  
- 🔐 Secure Authentication (Clerk)  
- ⚡ Modern Responsive UI  
- ☁️ Vercel Deployment

🚀 Getting Started

Clone Repository

git clone https://github.com/harshdwivediiiii/pathfinder-ai.git
cd pathfinder-ai

Install Dependencies

npm install

Setup Environment Variables

DATABASE_URL=your_postgresql_url

NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=your_key
CLERK_SECRET_KEY=your_secret

GEMINI_API_KEY=your_api_key

Run Development Server

npm run dev


---

🗄 Prisma Setup

npx prisma generate
npx prisma migrate dev
